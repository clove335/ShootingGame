## Installation
``` Bash
$ git clone https://github.com/clove335/ShootingGame.git
$ cd ShootingGame/
# If you have not installed Rust, you have to install it
$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
$ source ~/.bashrc
$ cargo build
$ cargo run
```
